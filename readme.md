# CYBERGLADE MAZE
A gesture-controlled maze game

## FEATURES
1. Gesture Control: Hands/Nose
2. Hover to interact: Player needs to hover over a button to interact with it
3. Single player/Local multiplayer: The maze can be played single player or compete with your friends (2 player), even need to work with your friend to finish specific map
4. Multiple difficulty: The maze has several map difficulty ranging from easy to hard or a special "Teamwork" map

## TECHS
1. Frontend: Vanilla JS, HTML5, CSS3
2. Computer Vision: Google MediaPipe
3. Fonts: Google Fonts

## HOW TO RUN?
1. Clone the this [repository](https://www.github.com/lamgu/cyberglade-maze.git)
2. Open VS Code
3. Open command pallete (Ctrl + Shift + P)
4. Choose "Live Server: Open with Live Server" (install the extension "Live Server" in "Extensions" tab, if you don't have it installed yet.)
5. Enjoy!
